4. Customizing Kuser

4.1 Edit user creation defaults

To edit user creation defaults you may use 'Edit defaults' dialog, which is accessible via menu as File->Preferences. There you may change defaults used to create new user: Shell, Base of Home Directory (or home directories volume), either create home directory on create or not, either copy skeleton to home directory, and use User Private Group mechanism (when creating user, his own group is created, also when removing user corresponding group removed also).
